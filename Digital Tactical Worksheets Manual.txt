ICS COMMAND BOARD - USER INSTRUCTION MANUAL
==============================================
Version: 2025 Command Board Release
Author: Daniel Romano

----------------------------------------------
1. OVERVIEW
----------------------------------------------
The ICS Command Board is a standalone, browser-based incident
command tool that allows drag-and-drop unit tiles, freehand
drawing, notes, multi-tab worksheets, custom backgrounds, and
layout saving/loading. It runs on desktops, tablets, iPads, and
can be installed as a fullscreen web app.

You can use this tool for structure fires, MCIs, high-rise
operations, wildland, training scenarios, and more.

3. HOW TO OPEN THE PROGRAM
----------------------------------------------

DESKTOP / LAPTOP:
-----------------
Option A – Local File
- Double-click index.html
- The program opens in your default web browser.

Option B – Installed Web App (recommended)
- Open your GitHub Pages URL
- In Chrome or Edge: menu → Apps → Install This Site as an App
- A desktop/start menu shortcut will appear.

iPAD / TABLET:
--------------
Option A – Safari Browser
- Open your GitHub Pages URL
- Use Fullscreen mode inside the app for best view.

Option B – Add to Home Screen (BEST)
- Tap Share → Add to Home Screen
- It launches fullscreen with no browser chrome.

----------------------------------------------
4. MAIN INTERFACE
----------------------------------------------

SIDEBAR (left side):
- Preset unit buttons (Engine, Tower, Rescue, RIT, Chiefs)
- Collapsible unit groups
- Notes panel (can be opened and moved)

WORKSHEET AREA (right side):
- Displays the currently selected tab’s background
- Allows tile dragging, drawing, checking, notes

TOOLBAR (top):
- Tile creation buttons
- Mode controls (Move / Draw / Erase / Check / Note)
- Utilities (Undo, Redo, Delete, Save Layout, Load Layout)
- Fullscreen button
- Timer controls
- “Change Worksheet” button

----------------------------------------------
5. TABS / MULTIPLE SHEETS
----------------------------------------------

The app starts with TWO tabs:

1. MAIN BOARD
   - Background: worksheet.png
   - Fully custom

2. BLANK SHEET
   - Background: blank_worksheet.png
   - Freeform layout

Each tab has:
- Its own tiles
- Its own background
- Its own drawing layer
- Its own note list
- Its own undo/redo history

To switch tabs:
- Tap/click the tab name at the top of the worksheet.

Tiles stay ONLY on the tab where they were created.

----------------------------------------------
6. UNIT TILES
----------------------------------------------

Creating a tile:
- Click any preset button (Engine, RIT, Tower, etc.)
- A color-coded tile appears on the worksheet.

Moving a tile:
- Click/tap + drag

Duplicating a tile:
- Double-click the tile

Deleting a tile:
- Select Delete mode, then tap the tile
OR
- Tap the Delete button with a tile selected

Long-press (mouse or touch):
- Opens NOTE editor for the tile

----------------------------------------------
7. TILE NOTES
----------------------------------------------

Adding/editing notes:
- LONG PRESS a tile (mouse: press & hold)
- Type note → OK
- A note icon appears on the tile
- The floating notes list updates automatically

Exiting back to Move Mode:
- Automatic after closing a note

Viewing notes:
- Open Floating Notes Window
- Tap any entry to jump to that tile

----------------------------------------------
8. DRAWING / MARKUP
----------------------------------------------

Draw mode:
- Allows freehand drawing on the worksheet

Erase mode:
- Erases ONLY drawing strokes (not tiles or image)

Checkmark mode:
- Single tap places a ✓ on the worksheet

Move mode:
- Default mode for dragging tiles

----------------------------------------------
9. BACKGROUND IMAGES
----------------------------------------------

The user can change the background on ANY tab:

1. Tap “Change BG”
2. Select ANY PNG/JPG/WebP file on your device
3. Worksheet updates instantly
4. Canvas resizes automatically
5. The custom background is saved in that tab

To restore defaults:
- Switch to Main Board or Blank Sheet (their default images load)
OR
- Reload a saved layout

----------------------------------------------
10. FULLSCREEN MODE
----------------------------------------------

Desktop/Laptop:
- Click “Fullscreen”
- Removes browser UI and enlarges worksheet

Exit fullscreen:
- Press ESC
- Or click “Exit Fullscreen”

iPad:
- Use Add to Home Screen for TRUE fullscreen
or
- Use the Fullscreen button inside the program

----------------------------------------------
11. SAVING & LOADING LAYOUTS
----------------------------------------------

Saving:
- Click “Save Layout”
- A .json file downloads containing:
  - Tiles
  - Notes
  - Drawings
  - Background images
  - ALL tabs and states

Loading:
- Click “Load Layout”
- Select a previously saved .json file

Layout fully restores exactly as it was.

----------------------------------------------
12. AUTO-STACK
----------------------------------------------

Auto-Stack organizes tiles on the CURRENT TAB:
- Sorts by type and groups similar units
- Helpful during large incidents

----------------------------------------------
13. MOBILE / TABLET OPTIMIZATION
----------------------------------------------

The program automatically adjusts:
- Sidebar moves to top on narrow screens
- Worksheet remains full width
- Buttons grow larger for finger input

For best experience:
- Use Landscape mode
- Use Fullscreen mode
- On iPad: install via Add to Home Screen

----------------------------------------------
14. PWA INSTALL (optional but recommended)
----------------------------------------------

Supported in Chrome, Edge, Safari (iPadOS 15+):

1. Visit your GitHub Pages URL
2. Browser menu → Install as App
3. Opens in standalone window with no URL bar
4. Works offline after first load

----------------------------------------------
15. TROUBLESHOOTING
----------------------------------------------

Tiles don’t move:
- Make sure you are in MOVE mode

Notes popup doesn't show:
- Long press without dragging
- Ensure you are not in DRAW mode

Background won’t change:
- Ensure PNG file is not corrupt

Overlapping UI on iPad:
- The responsive layout automatically fixes this
- Turn tablet to landscape

Program won’t install:
- Check manifest.json and service-worker.js exist
- Ensure you're using the HTTPS GitHub Pages URL

----------------------------------------------
16. VERSIONING & UPDATES
----------------------------------------------
